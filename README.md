# subhajeetdey.dev

My personal portfolio and blog, powered by a custom-built headless CMS.

## Tech Stack
- **Framework** — Next.js 14 (App Router)
- **API** — tRPC
- **Database** — PostgreSQL (Neon) + Prisma
- **Auth** — NextAuth.js
- **Deployment** — Vercel

## Features
- Admin CMS dashboard to manage posts and projects
- Markdown blog with draft/publish workflow
- SEO optimised with auto-generated sitemap and OG images

## Getting Started
\```bash
pnpm install
cp .env.example .env
pnpm dev
\```
